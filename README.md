# cobra-murat
CLI tool developed with Cobra


### Install 

```sh
go build . 
```

### Commands

###### kobra

```sh
./cobramurat kobra
```
###### play



```sh
./cobramurat play
```

###### 56

```sh
./cobramurat 56
```
